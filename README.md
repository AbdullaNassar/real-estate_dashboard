# Maskn Dashboard

**Maskn** is a modern, dynamic web application for managing home rentals, providing a dashboard for administrators to monitor statistics, manage properties, users, and bookings, with real-time data integration and secure authentication.


## [Live Demo](https://maskn-dashboard.netlify.app/)


## Features
- **Dynamic Dashboard**: Displays real-time statistics and charts powered by Supabase data
- **Admin Operations**: CRD (Create, Read, Delete) functionality for managing users, properties, and bookings.
- **Authentication**: Secure website authentication for admin access.
- **Responsive Design**: Built with Tailwind CSS for a seamless experience across devices.


##  Tech Stack
**Frontend**: React.js, Tailwind CSS, Redux Toolkit 

**Backend**: Supabase:

**Helper Libraries**: react query, recharts, react-hook-form, react-hot-toast, react icons

## Setup instructions

**Prerequisites** Node.js (v16 or higher) - npm

1- Clone the Repository

```bash 
git clone https://github.com/AbdullaNassar/real-estate_dashboard.git
cd real-estate-Dashboard 
```
2- Install Dependencies

Using npm:
```bash
npm install
```
3-Run the Development Server
```bash
npm run dev
```
The app will be available at http://localhost:5173 (or another port if specified).
